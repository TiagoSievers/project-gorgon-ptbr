# installer/

Artefato ativo do fluxo **pack-ptbr**:

| Arquivo | Uso |
|---------|-----|
| `copiar_installer_native.c` | Compilado → `dist/INSTALAR-COPIAR.exe` → `Instalador-Windows.exe` no pack |

Instaladores legados (PyInstaller, `INSTALAR` bash, `pack-windows`, etc.) estão em `deletar/installer/`.

Desinstalar: `scripts/uninstall.sh` ou skill Cursor (`DESINSTALAR-SKILL.md` no pack).
